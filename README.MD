## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/vbzt/Nomuz.git
cd Nomuz
```

Install dependencies for each module:

```bash
# API
cd api
npm install

# Blockchain
cd ../blockchain
npm install

# Frontend
cd ../frontend
npm install
```

### Running

```bash
# Start the API
cd api
npm run dev

# Start the Frontend
cd frontend
npm run dev
```
